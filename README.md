This repository contains laboratory exercises and computational assignments from the Discrete-Time Signal Processing course. It covers essential signal processing concepts, including discrete-time systems, filtering, frequency analysis, noise removal, and simulation methods using MATLAB.

Repository Contents

Exercise 2: Sampling and Chirp Signals

Converting continuous-time signals to discrete-time signals

Analysis and simulation of chirp signals

Practical implementation and audio playback using sound cards

Exercise 3: Digital Filtering and Noise Removal

Implementation and analysis of digital filters

Zero-Pole diagrams and frequency response analysis

Quantization of filter coefficients and its impact on filter performance

Practical noise removal from audio signals

Learning Objectives

Understand sampling, quantization, and practical implementation of digital signal processing concepts

Gain hands-on experience with MATLAB-based simulations and analyses

Analyze digital filters' performance in signal enhancement and noise removal

Tools and Software

MATLAB for simulation, signal analysis, and visualization
